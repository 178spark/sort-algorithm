各种排序规则
====

### 1.冒泡排序

基本思想：

两两比较待排序数据元素的大小，发现两个数据元素的次序相反（大的在前，小的在后）时即进行交换，每一次扫描会把扫描区间最大的数字放在尾部。

位置：com.algorithm.bubbling.BubblingSort
